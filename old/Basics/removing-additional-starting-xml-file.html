Removing Additional Starting File

These are steps to remove content_main.xml if you have both content_main.xml and activity_main.xml.

Step 1:
In activity_main.xml, there is a line that pulls in other xml page. Remove that line:
<include layout="@layout/content_main" />

Step 2:
Delete the content_main.xml file. You will lose the "Hello World" TextView. Readd it to activity_main.xml if you want.
